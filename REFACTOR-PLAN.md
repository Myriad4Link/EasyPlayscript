# PlayscriptBase Refactor — TDD Plan

## Goal

Replace the concrete `Registry` class with an abstract `PlayscriptBase` class.
All `interface` declarations in `.scpt` files become abstract methods.
User subclasses the generated base and implements the methods.
A configurable MSBuild property `<PlayscriptBaseClass>` controls the class name.

## Design Decisions

| Decision | Choice |
|---|---|
| Class kind | Abstract base class |
| User entry point | Subclass directly (no Registry) |
| Void interfaces | Abstract methods |
| Non-void interfaces | Abstract methods |
| Return values | Stored in `ConsumerCallItem.Result` |
| Arg extraction | Typed (e.g. `((StringArgument)call.Arguments[0]).Value`) |
| Class name | Configurable via `<PlayscriptBaseClass>` MSBuild property (default: `PlayscriptBase`) |
| Scope | One shared class from all `.scpt` files |

## Type Mapping

| `.scpt` interface | Generated abstract method |
|---|---|
| `interface transition(type: string) : void` | `abstract void transition(string type)` |
| `interface play(snd: string, vol: decimal) : void` | `abstract void play(string snd, double vol)` |
| `interface on_complete() : void` | `abstract void on_complete()` |
| `interface get_name() : string` | `abstract string get_name()` |
| `interface calc(a: int, b: int) : int` | `abstract int calc(int a, int b)` |
| `interface check(flag: bool) : void` | `abstract void check(bool flag)` |

## Generated Code Example

```csharp
// <auto-generated/>
using EasyPlayscript;
using EasyPlayscript.Generated;

public abstract class PlayscriptBase
{
    private readonly Lazy<Dictionary<string, ScriptBlock>> _scripts;
    private readonly Lazy<Dictionary<string, TextBlock>> _texts;

    protected PlayscriptBase(string outputPath, string aesKey)
    {
        _scripts = new Lazy<Dictionary<string, ScriptBlock>>(
            () => PlayscriptLoader.LoadScripts(outputPath, aesKey));
        _texts = new Lazy<Dictionary<string, TextBlock>>(
            () => PlayscriptLoader.LoadTexts(outputPath, aesKey));
    }

    public Script LOAD_TOOLTIP => new Script
    {
        Block = _scripts.Value["load_tooltip"],
        Dispatch = DispatchCall
    };

    public abstract void transition(string type);
    public abstract string get_name();

    internal void DispatchCall(ConsumerCallItem call)
    {
        switch (call.Identifier)
        {
            case "transition":
                transition(
                    ((StringArgument)call.Arguments[0]).Value);
                break;
            case "get_name":
                call.Result = get_name();
                break;
        }
    }
}
```
